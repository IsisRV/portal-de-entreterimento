# Meu Portal de Entretenimento Digital

Projeto desenvolvido para a atividade de CSS da ETEC, com o objetivo de aplicar conceitos de HTML e estilização utilizando CSS.

## Objetivo do Projeto

Criar uma página web temática sobre entretenimento digital, dividida em três categorias principais:

- Filmes e Séries
- Música e Podcasts
- Games e Tecnologia

O projeto foi desenvolvido para praticar:

- Estruturação de páginas com HTML
- Estilização com CSS
- Uso de CSS Inline, Interno e Externo
- Classes e IDs
- Layout com Flexbox
- Organização de arquivos web

---

## Tecnologias Utilizadas

- HTML5
- CSS3
- Flexbox

---

## Estrutura do Projeto

```txt
portal-entretenimento/
│
├── portal.html
├── estilo.css
├── reset.css
│
└── imagens/
```

### Arquivos do projeto

### `portal.html`
Arquivo principal da página web contendo a estrutura do portal.

### `estilo.css`
Arquivo responsável pela estilização principal do site.

### `reset.css`
Arquivo CSS Reset baseado no modelo de Eric Meyer, utilizado para padronizar estilos entre navegadores.

### `imagens/`
Pasta contendo as imagens utilizadas nas seções do portal.

## Layout da Página

A página foi organizada em três colunas utilizando **Flexbox**, permitindo melhor alinhamento e distribuição do conteúdo.

As seções do portal são:

### Filmes e Séries
Contém imagens e descrições de filmes de animação.

### Música e Podcasts
Apresenta músicas e conteúdos relacionados ao entretenimento musical.

### Games e Tecnologia
Exibe jogos digitais e informações sobre tecnologia no entretenimento.
